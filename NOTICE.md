# Notice

This repository contains a web gateway, Docker packaging, and documentation for
running the MUHAN legacy MUD server in a browser.

It does **not** vendor or relicense the MUHAN game source or data. By default,
the Docker build fetches the upstream restoration repository from:

- https://github.com/comfuture/muhan

Before hosting a public server, review the upstream project's copyright and
license terms, including any non-commercial-use restrictions inherited from the
original MUHAN source distribution.
